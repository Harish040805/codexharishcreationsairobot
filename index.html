Robot Workstation Instructions

The robot operates through a custom command-based language. All commands must be typed in the workstation input field and executed by pressing the Enter key. Commands are case-sensitive and must follow the exact formats shown below.

For best results, give values in the range of 10-50 for Move commands and in the range of 10-30 for Rotate commands.

NOTE: no means number; MOV UP(no) => MOV UP(give any number); Ex: MOV UP(5)
1. Start and Stop Commands
  • HELO – Initial command used to turn on the robot and make it work, only after which the robot listens to the user given command.
  • GOODBYE – Final command to turn the robot off.
2. Standard Movement Commands
  • MOV UP(no) – Moves the robot upward by the specified number of steps.
  • MOV DN(no) – Moves the robot downward by the specified number of steps.
  • MOV LF(no) – Moves the robot left by the specified number of steps.
  • MOV RT(no) – Moves the robot right by the specified number of steps.
3. Standard Rotation Commands
  • ROT LF(no) – Rotates the robot to the left by the given degree or unit.
  • ROT RT(no) – Rotates the robot to the right by the given degree or unit.
4. Infinite and Reset Commands
  • MOV UP(inf), MOV DN(inf), MOV LF(inf), MOV RT(inf) – Moves the robot infinitely in the specified direction until another command is given.
  • ROT LF(inf), ROT RT(inf) – Continuously rotates the robot in the given direction until interrupted.
  • MOV ORG – Instantly returns the robot to its original starting position.
  • STOP - Stop the robot in the last saved position during movement or rotation.
  • /NBR - Brings the code to the next line or creates a new line.
  • SPD no - Sets speed for the robot externally (optional). No () for setting speed.
5. Turbo Mode Commands

Turbo mode allows faster movement and rotation using the same set of commands. To activate turbo, use the prefixes TRB (for movement) or TROT (for rotation).

Turbo command examples:

  • TRB UP(no) – Moves the robot upward at turbo speed.
  • TRB DN(no) – Moves the robot downward at turbo speed.
  • TRB LF(no) – Moves the robot left at turbo speed.
  • TRB RT(no) – Moves the robot right at turbo speed.
  • TROT LF(no) – Rotates the robot left at turbo speed.
  • TROT RT(no) – Rotates the robot right at turbo speed.
6. Combined Movement and Rotation Commands

The robot can perform combined movement and rotation actions using the MOV ROT format. These commands cause the robot to move in a specified direction while simultaneously rotating either left or right. Infinite versions of these commands are also supported.

  • MOV ROT UP LF(no) – Moves upward while rotating left.
  • MOV ROT UP RT(no) – Moves upward while rotating right.
  • MOV ROT DN LF(no) – Moves downward while rotating left.
  • MOV ROT DN RT(no) – Moves downward while rotating right.
  • MOV ROT UP LF(inf), MOV ROT UP RT(inf) – Infinite combined motion upward with rotation.
  • MOV ROT DN LF(inf), MOV ROT DN RT(inf) – Infinite combined motion downward with rotation.

Turbo versions of these combined commands also exist. You can prefix them with TRB or TROT for turbo performance.

7. Workstation Behavior
  • The workstation contains a single input field where commands are entered.
  • Each command is executed by pressing the Enter key.
  • The STOP command is used to stop the infinitely moving robot in the last saved position.
  • There are no on-screen panels or buttons — all control is via commands.
  • If an invalid or unrecognized command is entered, an error message will be displayed in the workstation.
